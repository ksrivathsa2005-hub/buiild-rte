Quickstart

The best way to get started is to try a simple example. The Rich Text Editor is initialized with a DOM element to contain the editor. The contents of that element will become the initial contents of the editor.

HTML 
<!-- Include stylesheet -->
<link href="rte-package/src/styles/main.css" rel="stylesheet" />
<link href="rte-package/src/styles/components.css" rel="stylesheet" />

<!-- Create the editor container -->
<div id="editor">
    <p>Hello World!</p>
    <p>Some initial <strong>bold</strong> text</p>
</div>

<!-- Include the RTE library -->
<script type="module">
    import RTE from 'rte-package/src/index.js';
    import { DEFAULT_TOOLBAR } from 'rte-package/src/config/defaults.js';
    
    // Use DEFAULT_TOOLBAR - 90% configurable!
    const editor = new RTE('editor', {
        toolbar: DEFAULT_TOOLBAR.filter(g => g.group === 'formatting')
    });
    
    // Or use custom toolbar for more control
    const customEditor = new RTE('editor', {
        toolbar: [
            { group: 'formatting', items: [
                { type: 'button', label: 'Bold', command: 'bold', icon: '<i class="fas fa-bold"></i>' },
                { type: 'button', label: 'Italic', command: 'italic', icon: '<i class="fas fa-italic"></i>' }
            ]}
        ]
    });
</script>

And that's all there is to it!

Installation

Install the Nalashaa RTE package via npm:

Terminal 
npm install @nalashaa-sri/vanilla-rte

Or include directly via CDN:

HTML 
<link rel="stylesheet" href="https://unpkg.com/@nalashaa-sri/vanilla-rte/dist/styles/main.css">
<script type="module" src="https://unpkg.com/@nalashaa-sri/vanilla-rte/dist/index.js"></script>
Icon Dependencies

DEFAULT_TOOLBAR uses Font Awesome icons. Include Font Awesome in your project:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

You can customize icons or use your own icon library by modifying the icon property in toolbar items.

Configuration

The Rich Text Editor is 90% configurable, allowing you to customize nearly every aspect. All default configurations are defined in rte-package/src/config/defaults.js.

Recommended Approach

Import and use DEFAULT_TOOLBAR, DEFAULT_THEME, DEFAULT_FEATURES, and other defaults from config/defaults.js for easy setup. See CONFIGURATION_GUIDE.md for detailed examples.

Option Type Description
toolbar Array Array of toolbar groups with items (use DEFAULT_TOOLBAR)
theme Object Theme configuration with colors, fonts, spacing (use DEFAULT_THEME)
features Object Feature toggles for undo, spellcheck, etc. (use DEFAULT_FEATURES)
i18n Object Internationalization strings (use DEFAULT_I18N)
editor Object Editor options like placeholder, readOnly (use DEFAULT_EDITOR_OPTIONS)

Toolbar Overview

The Rich Text Editor provides a powerful toolbar that enables users to format, style, and edit content efficiently. The toolbar includes essential editing tools such as bold, italic, underline, alignment, and lists, along with customization options to suit different use cases.

Best Practice

Use DEFAULT_TOOLBAR from config/defaults.js and filter/customize as needed:

import { DEFAULT_TOOLBAR } from 'rte-package/src/config/defaults.js';

// Use only specific groups
const editor = new RTE('editor', {
  toolbar: DEFAULT_TOOLBAR.filter(g => ['formatting', 'paragraph'].includes(g.group))
});

Toolbar Types

The Rich Text Editor allows you to configure different types of toolbar using the type field in toolbarSettings property. The types of toolbar are:

  • Expand — Default mode, hides overflow in expandable section
  • MultiRow — Displays toolbar items in multiple rows
  • Scrollable — Horizontal scrolling for overflow items

Expand (Default)

The default mode of the toolbar is Expand. This mode wraps any overflowing toolbar items to the next row, allowing all tools to remain visible.

JavaScript 
import { DEFAULT_TOOLBAR } from 'rte-package/src/config/defaults.js';

const editor = new RTE('editor', {
    toolbar: DEFAULT_TOOLBAR.filter(g => ['clipboard', 'formatting'].includes(g.group))
});
Expand Toolbar Demo

MultiRow Toolbar

The MultiRow mode displays all toolbar items across multiple rows, ensuring every tool is immediately visible without scrolling or expanding.

JavaScript 
import { DEFAULT_TOOLBAR } from 'rte-package/src/config/defaults.js';

// Use full toolbar with all groups
const editor = new RTE('editor', {
    toolbar: DEFAULT_TOOLBAR
});
MultiRow Toolbar Demo

Scrollable Toolbar

The Scrollable mode keeps all toolbar items in a single row with horizontal scrolling for overflow. This is useful for compact layouts.

Tip

Use CSS to enable horizontal scrolling: .rte__toolbar { flex-wrap: nowrap; overflow-x: auto; }

CSS 
/* Scrollable toolbar style */
.rte__toolbar {
    flex-wrap: nowrap;
    overflow-x: auto;
    -webkit-overflow-scrolling: touch;
}

.rte__toolbar::-webkit-scrollbar {
    height: 6px;
}

.rte__toolbar::-webkit-scrollbar-thumb {
    background: #ccc;
    border-radius: 3px;
}

Content API

Methods for getting and setting editor content:

JavaScript 
// Get HTML content
const html = editor.getContent();

// Set HTML content
editor.setContent('<h1>New Content</h1><p>Hello World</p>');

// Clear content
editor.clearContent();

// Load from JSON
editor.loadFromJSON({ content: '<p>Loaded content</p>' });

// Load from HTML file
editor.loadFromHTML(htmlString);

Formatting API

Execute formatting commands programmatically:

JavaScript 
// Execute a command
editor.execute('bold');
editor.execute('foreColor', '#ff0000');
editor.execute('formatBlock', 'h2');

// Check if command is active
const isBold = editor.queryCommandState('bold');

Events

Listen to editor events:

JavaScript 
// Content change event
editor.on('content-change', (content) => {
    console.log('Content changed:', content);
});

// Selection change event
editor.on('selection-change', (selection) => {
    console.log('Selection:', selection);
});

Themes

Customize the editor appearance using CSS variables:

CSS 
:root {
    --color-primary: #333333;
    --color-primary-dark: #1a1a1a;
    --color-primary-light: #f5f5f5;
    --color-bg: #ffffff;
    --color-border: #e0e0e0;
    --color-text: #2c3e50;
    --color-text-light: #6c757d;
    --font-family-base: 'Segoe UI', sans-serif;
}

/* Dark theme example */
.rte.dark-theme {
    --color-bg: #1a1a1a;
    --color-text: #e0e0e0;
    --color-border: #333;
}

Quickstart

The best way to get started is to try a simple example. The Rich Text Editor is initialized with a DOM element to contain the editor.

HTML 
<!-- Include stylesheet -->
<link href="node_modules/editor-structure/dist/styles/main.css" rel="stylesheet" />

<!-- Create the editor container -->
<div id="editor">
    <p>Hello World!</p>
</div>

<!-- Include the RTE library -->
<script type="module">
    import RTE from 'editor-structure';
    
    // Initialize with default settings
    const editor = new RTE('editor');
</script>

Installation

Install the Nalashaa RTE package via npm:

Terminal 
npm install editor-structure

Or include directly via CDN:

HTML 
<link rel="stylesheet" href="https://unpkg.com/editor-structure/dist/styles/main.css">
<script type="module" src="https://unpkg.com/editor-structure/dist/index.js"></script>

Configuration

The Rich Text Editor is highly configurable. You can customize the toolbar, theme, and enabled features by passing an options object to the constructor.

JavaScript 
const editor = new RTE('editor-container', {
    placeholder: 'Start typing...',
    disabled: false,
    toolbar: [ /* Custom toolbar config */ ]
});

Toolbar Customization

You can completely customize the toolbar by passing an array of group objects. Each group contains a list of items (buttons, selects, color pickers). Below is a complete list of available tools.

1. Basic Formatting

Standard text formatting options.

Bold bold Makes text bold
Italic italic Italicizes text

Strikethrough strikeThrough Crosses out text
Superscript superscript Raises text (e.g., x²)
Subscript subscript Lowers text (e.g., x₂)
Inline Code code Formats text as code snippet
Clear Format clearFormatting Removes all styling
Export export Export content as JSON, HTML, or PDF

Outdent outdent Decrease indentation

Toolbar Layouts

MultiRow Toolbar (Default)

The default toolbar automatically wraps items to multiple rows if they exceed the container width.

JavaScript
import { DEFAULT_TOOLBAR } from 'editor-structure';

const editor = new RTE('editor', {
    toolbar: DEFAULT_TOOLBAR // Uses all default groups
});

Scrollable Toolbar

The Scrollable mode keeps all toolbar items in a single row with horizontal scrolling for overflow. This is useful for compact layouts.

Tip

Use CSS to enable horizontal scrolling:

CSS
.rte__toolbar {
    flex-wrap: nowrap;
    overflow-x: auto;
    -webkit-overflow-scrolling: touch;
}
.rte__toolbar::-webkit-scrollbar { height: 6px; }
.rte__toolbar::-webkit-scrollbar-thumb { background: #ccc; border-radius: 3px; }

Content API

Methods for getting and setting editor content:

JavaScript
// Get HTML content
const html = editor.getContent();

// Set HTML content
editor.setContent('
New Content
Hello World
');

// Clear content
editor.clearContent();

// Load from JSON
editor.loadFromJSON({ content: '
Loaded content
' });

// Load from HTML string
editor.loadFromHTML(htmlString);

Formatting API

Execute formatting commands programmatically:

JavaScript
// Execute a command
editor.execute('bold');
editor.execute('foreColor', '#ff0000');
editor.execute('formatBlock', 'h2');

// Check if command is active
const isBold = editor.queryCommandState('bold');
